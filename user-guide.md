# カフェヨシノ 本社カレンダー 使い方

## 予定を入れる人がやること

1. Googleカレンダーを開く
2. 予定を追加する
3. カレンダー欄で「カフェヨシノ本社行動予定表」を選ぶ
4. タイトルを入れる
5. 時間を入れる
6. 保存する

タイトルに自分の名前を入れる必要はありません。
アプリ側で、予定を作ったGoogleアカウントから名前を表示します。

## 管理する人がやること

1. 各メンバーのGmailを共有カレンダーに招待する
2. 権限を「予定の変更」にする
3. `app-config.js` にメールと名前の対応表を入れる

例：

```text
imanishi0305@gmail.com=今西
tamaki@example.com=玉木
nakata@example.com=中田
```

## 注意

予定を入れるときに個人カレンダーを選ぶと、本社カレンダーアプリには表示されません。
必ず「カフェヨシノ本社行動予定表」を選んでください。
