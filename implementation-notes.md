# 本社カレンダー 実装メモ

## できていること

- 日別、週別、月別の表示
- メンバー別の絞り込み
- 予定追加フォーム
- Googleカレンダー設定フォーム
- Google Calendar APIからの予定取得
- Google Calendar APIへの予定追加
- Googleカレンダー予定名から担当者名を読み取り
- スマホのホーム画面追加用の設定
- 5分ごとの自動同期の下準備

## Googleカレンダー側の予定名ルール

共有カレンダーに直接入れる予定は、タイトルを次の形にします。

```text
担当者：内容
```

例：

```text
玉木：店舗訪問
中田：業者対応
```

この形なら、アプリ側で担当者名を自動で分けて表示します。

タイトルに担当者名を入れない場合は、アプリの設定画面に「メールと名前の対応表」を入れます。

```text
imanishi0305@gmail.com=今西
president@example.com=社長
```

この場合、Googleカレンダーの予定を作成したアカウントのメールアドレスから担当者名を表示します。

## 本番化で必要な作業

1. 本社用Googleアカウントで共有カレンダーを作る
2. 本社メンバーに共有カレンダーの編集権限を付ける
3. Google CloudでGoogle Calendar APIを有効化する
4. APIキーを作る
5. OAuthクライアントIDを作る
6. OAuthの承認済みJavaScript生成元に、実際に置くURLを登録する
7. アプリの設定画面に、共有カレンダーID、APIキー、OAuthクライアントIDを入力する

## 置き場所について

本番では `http://127.0.0.1` ではなく、社内用のURLに置くのがおすすめです。
たとえば `https://calendar.example.com` のようなURLです。

OAuth設定には、そのURLを登録する必要があります。
